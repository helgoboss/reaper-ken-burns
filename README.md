REAPER 5 slideshow generator with Ken Burns effect
==================================================

Instructions
------------
1. Set "File => Project settings => Video => Preferred
   video size" to a size of your choice, for example
   1920 x 1080
2. Add a track
3. If you want fades, add FX "Video processor" to the 
   track FX chain with preset "Item fades affect video"
4. Add image/video items to the track (order matters,
   exact positions and lengths don't)
5. Select the track
6. Run this script
7. If you want to regenerate with different settings, 
   adjust parameters in script and rerun the script
8. You can fine tune the generated result by adjusting
   envelopes, item positions, item lengths etc.

Known issues
------------
- Doesn't work if you add any another take envelope to 
  the image/video items
